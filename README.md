# minimart app

Minimart App adalah frontend untuk aplikasi minimarket yang berinteraksi dengan Minimart Service. Aplikasi ini menampilkan kategori, produk, keranjang belanja, dan memungkinkan pengguna melakukan checkout (status order = pending).

## Cara Menjalankan Aplikasi

Install dependencies

```bash
 npm install
```

Copy file contoh .env.local

```bash
  cp env.local.example .env.local
```

Update data file .env.local

```bash
Buka file .env.local dan sesuaikan dengan BASE URL API backend (Minimart Service) yang sedang berjalan
```

jalankan server

```bash
 npm run dev
```
